unsigned int genRandNum ( unsigned int  seed_input)

Generates a pseudo-random number in the range [0, 9999]. If the input seed is negative or repeated, the function reuses the last known random state.

Parameters
seed_inputThe input seed used to initialize randomness. If seed_input < 0 or equals the previous seed, the last valid random state is used instead.
Returns
A pseudo-random integer in the range [0, 9999].
