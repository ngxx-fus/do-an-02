Detailed Description

16-bit Parallel Communication Driver Implementation

Implements "Safe-Mode" bus handling: Data bus is always kept as INPUT unless a WRITE operation is strictly performing.

Author
Nguyen Thanh Phu

P16Data_t Data )

Write a single word to the bus.

Writes a single 16-bit value to the bus.

  1. Prepare Mask for Data Pins
  1. SWITCH TO OUTPUT (Push-Pull) just for writing
  1. Calculate Bit Masks
  1. Drive Data to Pins
  1. Strobe Write (Low -> High)
  1. SAFETY: Switch back to INPUT (High-Z) immediately
