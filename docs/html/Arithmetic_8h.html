uint32_t GenerateRandomNumber ( uint32_t SeedInput)

Generates a pseudo-random number.

If the input seed is negative, the function reuses the last known random state. Otherwise, it reseeds the generator with the new value.

Parameters
SeedInputThe input seed. Pass a negative value (e.g., -1) to continue the sequence.
Returns
A pseudo-random integer in the range [0, 9999] (or customized range).
