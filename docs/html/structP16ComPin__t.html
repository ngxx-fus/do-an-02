Detailed Description

Pinout structure for 16-bit parallel communication.

Uses anonymous unions to allow access via specific names (e.g., .Read) or via arrays (e.g., .CtlPinArr[i]) for bulk configuration. Memory is organized into two contiguous blocks: Control and Data.
