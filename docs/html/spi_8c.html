def destroySPIDevice ( spiDev_t **  pDev)

Destroy and free all resources associated with an SPI device. This function disables interrupts, releases GPIO pins, clears all internal pointers, and frees the device structure.

Parameters
pDevDouble pointer to the spiDev_t object. After this call, *pDev will be set to NULL.
Returns
Default return status.

— 1. Uninstall ISR (only if in slave mode) —

— 2. Reset all GPIO pins to default (input floating) —

— 3. Clear buffers and internal pointers —

— 4. Reset other parameters —

— 5. Free memory and nullify pointer —
